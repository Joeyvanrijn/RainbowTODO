Actions

Add todo

The add todo functionality adds a todo to the todo list, depending on the input in the form.

Remove todo

The remove todo removes a todo from the list.

Sort todo's

The sort buttons sort the todo list by priority, add date or due date.

Check todo

This action marks a todo as done, and stripes the todo

Update todo

Update the todo's content

Catagorize todo's

Update the todo's catagory

OOP Description

The application wil consist out of the following items:
  • A module for the todolist, containing all the functions needed to manage, sort, show en edit the todolost. This module returns the functions that are available for the main variable
  • A main variable that will be fired after loading the document. this variable contains all the listners for the actions. For the keyboard controll we only needed to add the "return" listner, in order to add a todo. The other actions are accesable via the default keyboard navigation of the browser.
